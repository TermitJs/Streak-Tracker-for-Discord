TikTok-style chat streaks for Discord! Write every day → get a fire streak in DMs & profile. Auto-reset after 24h. Inspired by TikTok. Lightweight, open-source. Follow me: [GitHub](https://github.com/TermitJs) | [Donate](https://boosty.to/termitjs) | [Discord](https://discord.gg/DcqV7MS6nS)
# Streak Tracker for Discord

**TikTok-style chat streaks — now in Discord!**

> Write every day → get a **fire streak**  
> Miss a day → streak resets  
> Just like in **TikTok**, but for your DMs & servers

![Streak in DM](screenshots/dm.png)

## Usage

### Downloading

```bash
>> git clone https://github.com/TermitJs/Streak-Tracker-for-Discord
Or download ZIP from Releases

Installation (BetterDiscord)
bash1. Place files in:
   %AppData%\BetterDiscord\plugins\StreakTracker\
   
   ├── StreakTracker.plugin.js
   └── StreakTracker.config.js

2. Restart Discord
3. Settings → Plugins → Enable "Streak Tracker"

Starting
No setup needed!
Just write a message — streak starts automatically.
